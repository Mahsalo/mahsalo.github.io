RAG-based AI Research Assistant

Developed a RAG system to retrieve and summarize AI research papers. Built a vector search pipeline using FAISS and ChromaDB to enhance document retrieval. Integrated GPT-4/Mistral-7B for context-aware summarization, improving answer accuracy.

LLM Hallucination Detection with Parameter-Efficient Fine-Tuning

Built LLM hallucination detector using LoRA fine- tuning on Qwen3-4B. Identified domain transfer failure (QA-trained model: 0.99→0.28 F1 on summarization). Fixed via multi- domain training, achieving 0.98 F1 (QA) and 0.65 F1 (summarization) with only 0.3% trainable parameters.
